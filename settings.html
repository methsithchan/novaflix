Performance Mode

Optimize the app for low-end devices and Windows laptops.

Performance Mode Status

Device Score: /100

Auto: Automatically enables performance mode on low-end devices.
Enabled: Always use performance mode (recommended for Windows laptops).
Disabled: Always use full visual effects.

What Performance Mode Does

  • Disables heavy backdrop-blur effects (Windows performance killer)
  • Automatically disables particle animations
  • Reduces animation complexity and transitions
  • Simplifies shadows and visual effects
  • Optimizes scrolling for 60fps on low-end devices
